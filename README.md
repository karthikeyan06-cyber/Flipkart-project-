# Flipkart Project - ML Model Backup

This repository contains ML models and related code for the Flipkart project.

## Project Structure
- `models/` - Trained ML models
- `notebooks/` - Google Colab notebooks and experiments
- `data/` - Dataset files
- `scripts/` - Python scripts for training and inference

## Getting Started
1. Clone the repository
2. Install required dependencies
3. Load and use the trained models

## Note
Models are backed up from Google Colab environment.
